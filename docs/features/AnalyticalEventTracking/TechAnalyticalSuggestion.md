# Technical PRD: Internal Founder Analytics

Version: 1.0

Branch:

```text
feature/internal-founder-analytics
```

---

# Objective

Implement a privacy-safe internal analytics platform for RollFinders.

The system SHALL measure:

* Marketplace growth
* Visitor activity
* Search activity
* Academy engagement
* Commercial intent
* Claim funnel performance

The system SHALL NOT depend on third-party analytics providers.

---

# Architecture

```text
Browser
    ↓
Analytics API
    ↓
Analytics Service
    ↓
analytics_events

analytics_network_visits

analytics_daily_metrics
    ↓
Founder Dashboard
```

Analytics failures SHALL NOT impact user journeys.

# Database

## analytics_events

Purpose:

Store immutable analytics events.

```sql
CREATE TABLE analytics_events (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

  event_name VARCHAR(255) NOT NULL,

  visitor_id VARCHAR(100) NULL,

  session_id VARCHAR(100) NULL,

  ip_hash TEXT NULL,

  academy_id UUID NULL,

  open_mat_id UUID NULL,

  source VARCHAR(100) NULL,

  metadata_json JSONB NOT NULL DEFAULT '{}'::jsonb,

  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

Indexes:

```sql
CREATE INDEX idx_analytics_events_name_created
ON analytics_events(event_name, created_at);

CREATE INDEX idx_analytics_events_visitor_created
ON analytics_events(visitor_id, created_at);

CREATE INDEX idx_analytics_events_ip_hash_created
ON analytics_events(ip_hash, created_at);

CREATE INDEX idx_analytics_events_academy
ON analytics_events(academy_id);

CREATE INDEX idx_analytics_events_open_mat
ON analytics_events(open_mat_id);
```

---

## analytics_network_visits

Purpose:

Track anonymous network activity.

```sql
CREATE TABLE analytics_network_visits (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

  ip_hash TEXT NOT NULL UNIQUE,

  first_seen_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),

  last_seen_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),

  total_visit_count INTEGER NOT NULL DEFAULT 1,

  total_event_count INTEGER NOT NULL DEFAULT 0,

  suspected_bot BOOLEAN NOT NULL DEFAULT FALSE,

  metadata_json JSONB NOT NULL DEFAULT '{}'::jsonb,

  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),

  updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

---

## analytics_daily_metrics

Purpose:

Store aggregated dashboard metrics.

```sql
CREATE TABLE analytics_daily_metrics (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

  metric_name VARCHAR(255) NOT NULL,

  metric_value INTEGER NOT NULL,

  metric_date DATE NOT NULL,

  metadata_json JSONB NOT NULL DEFAULT '{}'::jsonb,

  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),

  UNIQUE(metric_name, metric_date)
);
```

# Analytics Identity Model

## visitor_id

Source:

```text
rf_visitor_id cookie
```

Purpose:

* Unique visitors
* Returning visitors

Rules:

* Generated by application
* Anonymous
* No personal data

---

## session_id

Source:

```text
rf_session_id cookie
```

Purpose:

* Session tracking
* Session analytics

Rules:

* Generated by application
* Multiple sessions may belong to one visitor

---

## ip_hash

Source:

```text
SHA256(ip + salt)
```

Purpose:

* Network analytics
* Bot detection
* Traffic quality

Rules:

* Raw IP MUST NOT be stored
* ip_hash MUST NOT be displayed in UI
