# PRD: About Page

Route: `/about`

Source: `src/app/about/page.tsx`

## Purpose

Explain what RollFinder is and who it serves.

## Requirements

IF a public user opens `/about`  
WHEN the page renders  
THEN it SHALL explain the platform, BJJ community focus, and mission.

IF the page renders  
WHEN metadata is evaluated  
THEN it SHOULD include appropriate title and description.

## Done When

* Page works without authentication.
* Content is mobile friendly.
* Footer links remain visible.
