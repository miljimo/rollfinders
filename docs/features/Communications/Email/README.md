# Email PRDs

This folder groups outbound email, email provisioning, and user account email requirements.

## Feature Areas

### Operations

Email system enablement and queue operations:

* `Operations/APIs/JobsEmailDeliveryApi.md`
* `Operations/APIs/AdminEmailProvisioningApi.md`
* Deployment provisioning: `../../Deployment/Provisioning/SesDomainVerificationPrd.md`

### User Account Emails

All-user password reset and admin-created user onboarding emails:

* `UserAccountEmails/APIs/UserPasswordResetApi.md`
* `UserAccountEmails/Products/AdminUserOnboardingEmailDeliveryPrd.md`
* `UserAccountEmails/Products/AdminUserOnboardingCredentialContentPrd.md`

## Ownership Notes

Email PRDs remain linked from user, platform, and MVP planning documents where those workflows queue email, but this folder is the canonical home for email-specific delivery and enablement requirements.
