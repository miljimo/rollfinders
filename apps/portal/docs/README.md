# Portal Documentation

This folder contains documentation owned by the Next.js portal.

Use this tree for:

- public site pages and route requirements
- dashboard pages and workflows
- reusable UI component PRDs
- portal-owned API route contracts
- mobile web experience docs
- frontend service-integration tickets
- UI assets and mockups

Backend service product contracts, service APIs, migrations, and backend runbooks live beside each backend service under:

```text
apps/backend_api/internal/services/<service>/docs/
```

Shared deployment, platform strategy, infrastructure, and cross-cutting product docs remain under top-level `/docs`.
