# PRD: Contact Page

Route: `/contact`

Source: `src/app/contact/page.tsx`

## Purpose

Provide contact paths for users, academy owners, partners, and support.

## Requirements

IF a public user opens `/contact`  
WHEN the page renders  
THEN it SHALL show contact/support/business enquiry information.

IF contact links are present  
WHEN activated  
THEN they SHALL use valid mail or navigation targets.

## Done When

* Page works without auth.
* Contact routes are visible.
* Layout is responsive.
