# Users Service Docs

This folder contains documentation owned by the users service.

- `product.md`: current service product contract.
- `prds/`: PRDs and proposals tied to this service.
- `tickets/`: service-owned implementation tickets.
- `runbooks/`: operational guidance.
- `api/`: OpenAPI and API contract docs.
- `architecture/`: service architecture notes.
- `archive/`: stale, superseded, or unimplemented docs.

Shared platform, deployment, UI, and cross-service docs remain under `/docs`.
